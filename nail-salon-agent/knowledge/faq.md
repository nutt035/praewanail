# คำถามที่พบบ่อย

ถาม: ต้องจองก่อนมั้ย?
ตอบ: จองล่วงหน้าเท่านั้นค่ะ รับคิว จันทร์ - ศุกร์ : 17:00 - 20:00 น. เสาร์ - อาทิตย์: 11:00 - 20:00 น. แต่อาจจะมีการเปลี่ยนแปลงได้ ควรสอบถามคิวอีกครั้งค่ะ

ถาม: รับ walk-in ไหม?
ตอบ: รับค่ะ แต่ลูกค้าที่จองคิวมาก่อนจะได้คิวก่อน ถ้าไปถึงหน้าร้านแล้วไม่มีคิวหรือไม่สะดวกก็จะต้องจองเป็นคิววันอื่นแทนค่ะ

ถาม: มีที่จอดรถไหมคะ?
ตอบ: มีที่จอดรถค่ะ

ถาม: ร้านอยู่ที่ไหนคะ?
ตอบ: หมู่บ้านพฤกษา บี คลองสามค่ะ หากลูกค้าจะเข้ามาใช้บริการ สามารถสอบถามเส้นทางเพิ่มเติมได้นะคะ

ถาม: ใช้เวลาทำนานไหมคะ?
ตอบ: ถ้าจองคิวมาแล้วประมาณ 1-2 ชั่วโมง แต่ถ้าไม่ได้จองคิวอาจจะต้องรอค่ะ

ถาม: ถ้าทำแล้วไม่ถูกใจสามารถแก้ได้ไหมคะ?
ตอบ: หากทำแล้ว สีเจลหรือลวดลายเสียหายสามารถแก้ได้ค่ะ แต่ต้องอยู่ภายใน 3 วันหลังทำ และต้องเป็นกรณีที่เกิดจากความผิดพลาดของทางร้าน
