# marketing.md — Marketing & Content Strategy

---

## Brand Voice

**Personality:** Like a knowledgeable friend who's great at nails — warm, trustworthy, not corporate.

| Do | Don't |
|----|-------|
| Warm, friendly, professional | Overly formal |
| Confident recommendations | Hard selling |
| Clean, feminine language | Excessive slang |

---

## Instagram Content Mix

| Type | % of posts | Examples |
|------|-----------|---------|
| Portfolio / Before & After | 40% | finished sets, transformation shots |
| Behind the scenes | 25% | process videos, tools, workspace |
| Education / Tips | 20% | nail care tips, trend explanations |
| Promotions | 15% | limited slots, seasonal offers |

### Caption Formula
```
[Hook — first line must stop the scroll]
[2–3 lines of value or story]
[CTA — book via link in bio / DM us]
.
.
#hashtags
```

### Core Hashtags
```
#nailart #naildesign #gelnails #nailsofinstagram
#[cityname]nails #[shopname]
```

## Template Sales Strategy

### Target buyers
- Nail salon owners who need a system but can't code
- Newly opened salons looking for a complete setup
- Freelance developers who build sites for salons

### Pricing tiers (draft)
```
Starter    — THB X,XXX  — code + setup guide (DIY)
Pro        — THB XX,XXX — code + done-for-you setup + 1 month support
Agency     — THB XX,XXX — white-label license for resellers
```

### Sales channels
- Facebook groups (nail technician + salon owner communities)
- TikTok — demo video showing what the system does
- Gumroad or Lemon Squeezy for digital product delivery
